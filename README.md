# ClientModPackToServer

Локальная CLI-утилита на Node.js для сборки серверной версии Minecraft-модпака из клиентской без изменения исходной папки `mods`.

В публичной части репозитория лежат только исходники самого инструмента: точка входа `index.js`, основная логика в `src/`, локальные данные в `data/` и описание зависимостей в `package.json`. Рабочие артефакты вроде `build/`, `cache/`, `reports/`, `tmp/`, а также локальная внутренняя документация и тестовые служебные каталоги в репозиторий не входят.

Сейчас проект уже умеет читать metadata из `.jar`, прогонять несколько движков классификации, учитывать зависимости, использовать registry, делать deep-check спорных модов и выполнять post-build smoke-test. Это именно кодовая база инструмента, а не готовая серверная сборка.
